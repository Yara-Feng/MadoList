# MadoList 部署指南

## 前置条件

1. 微信开发者工具（最新版）
2. 已开通微信云开发环境
3. Node.js 环境（用于云函数）

## 部署步骤

### 1. 导入项目

打开微信开发者工具 → 导入项目 → 选择 `f:/WeiXin/MadoList` 目录

### 2. 配置云环境

确保 `miniprogram/app.js` 中的 `env` 值与你的云环境 ID 一致。

### 3. 创建数据库集合

在云开发控制台 → 数据库 → 创建以下集合：

- `users`
- `groups`
- `tasks`
- `reminders`
- `notifications`
- `checkins`

### 4. 配置索引

在云开发控制台为每个集合创建索引（参考 `规范化记录.md` 3.3 章节）。

### 5. 配置安全规则

在云开发控制台 → 数据库 → 安全规则，为每个集合配置：
- 用户只能读写自己的数据
- 小组共享数据组员可读

### 6. 部署云函数

在微信开发者工具中，右键每个云函数目录（`cloudfunctions/login`、`user`、`group`、`task`、`reminder`、`notification`）→ **上传并部署：云端安装依赖**。

### 7. 准备 TabBar 图标

准备 8 个 81x81 px 的 PNG 图标文件放入 `miniprogram/images/tab/` 目录（参见该目录下的 README.md）。

### 8. 编译运行

点击微信开发者工具的「编译」按钮，即可在模拟器中预览。
